# eth-todo_list

## project setup

  1. install dependencies
      ```properties
      $ npm i -g truffle
      $ npm ci 
      ```
  2. install environment
  
       - [Ganache - Truffle Suite](https://trufflesuite.com/ganache/)
       - [MetaMask | The crypto wallet for Defi, Web3 Dapps and NFTs](https://metamask.io/)

## truffle cli command

  - `truffle compile`
  - `truffle migrate`
  - `truffle console`
